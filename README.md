# Portfolio

Simple personal portfolio website containing a single static page built with HTML, CSS and JavaScript.

## Features
- Clean, responsive layout
- Sections styled with CSS files in the `css/` folder
- Small interactive behavior in `js/main.js`

## Project structure

- index.html — main page
- css/ — styles (base.css, sections.css, style.css)
- js/ — JavaScript (main.js)

## Local preview

Quick preview by opening `index.html` in your browser (double-click).

Or run a simple local server (requires Python):

```powershell
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Editing

- Update HTML in `index.html`.
- Modify layout and colors in the files under `css/`.
- Add or change behavior in `js/main.js`.

## Deployment

This is a static site — you can deploy it on GitHub Pages, Netlify, Vercel, or any static host by pointing the host to this repository's root.
